## Instalation
- Clone the repository
```
git clone https://github.com/nikitunkun/WebSocketGame.git
or
git@github.com:nikitunkun/WebSocketGame.git
```
- Navigate to the project folder
```
cd WebSocketGame
```
- Run main.py
```
python3 backend/main.py
or
make run
```
